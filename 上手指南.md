# Idle Chronicles 上手指南

## 玩家篇 — 怎么玩？

1. 浏览器打开 **http://47.95.252.163:3001**
2. 输入用户名和密码，点「注册」
3. 登录后点 **「▶ 开始挂机」** 就开始自动打怪了
4. 底部四个标签：战斗 / 背包 / 属性 / 系统，各管各的

### 快速上手

| 你想…… | 怎么做 |
|---------|--------|
| 打怪升级 | 点「开始挂机」，自动战斗，关掉浏览器也会继续 |
| 穿装备 | 切到「背包」标签 → 点装备后的「装备」按钮 |
| 强化装备 | 背包页 → 装备槽下方点「强化」 |
| 学技能 | 战斗页 → 主动技能点「自动」让其自动释放 |
| 提升属性 | 属性页 → 属性修炼 点「提升」或「MAX」 |
| 爬塔 | 系统页 → 无尽之塔选择层数 |
| 每日签到 | 系统页 → 每日签到点「领取」 |
| 转生 | 系统页 → 达到 50 级后点「转生」获得永久加成 |

## 管理员篇 — 服务器维护

### 查看运行状态

```bash
pm2 status
```

### 重启服务（改了代码后）

```bash
cd ~/idle-rpg
git pull origin master            # 拉取最新代码
cd server && npm install && cd .. # 更新依赖
cd client && npm install && npx vite build && cd ..  # 重新构建前端
pm2 restart idle-chronicles       # 重启
```

### 常用 PM2 命令

```bash
pm2 status          # 查看状态
pm2 logs            # 查看日志（调试用）
pm2 restart all     # 重启所有服务
pm2 stop all        # 停止所有服务
pm2 monit           # 实时监控面板
```

### 数据备份

所有玩家数据在 `~/idle-rpg/server/storage/` 目录：

```bash
# 打包备份
tar -czf idle-backup-$(date +%Y%m%d).tar.gz ~/idle-rpg/server/storage/

# 下载到本地（在你自己电脑上执行）
scp root@47.95.252.163:~/idle-backup-*.tar.gz .
```

建议每周备份一次。

### 改域名 / HTTPS

编辑 `/etc/nginx/sites-available/idle-chronicles`，替换 `server_name`，然后：

```bash
sudo apt-get install -y certbot python3-certbot-nginx
sudo certbot --nginx -d 你的域名
```

## 开发篇

### 本地运行

```bash
git clone https://github.com/Li-Hee/idle-rpg.git
cd idle-rpg

# 后端
cd server
npm install
JWT_SECRET=本地测试密钥 node src/index.js

# 前端（新开一个终端）
cd client
npm install
npx vite
```

打开 http://localhost:3000 开发调试，API 自动代理到后端 3001 端口。
